## 1、事件
事件指的是某种情况的陈述。在概率论中，事件可以理解为某种试验或随机现象的结果。例如，掷骰子得到的点数是一个事件，可以定义为掷出偶数点数的事件A = {2, 4, 6}。每个具体结果（如掷出2点）称为基本事件，一个事件由若干基本事件组成。因此，事件本质上是一个集合。

### （1）事件之间的关系
由于事件是集合，因此我们可以用集合运算符来表达事件之间的逻辑关系。以下是一些基本关系：

- **蕴含与相等**：如果事件A发生必然导致事件B发生，则记作 $A \subseteq B$。如果事件A和B互相蕴含，则称A和B相等，记作 $A = B$。
- **互斥与对立**：互斥事件指在一次试验中不可能同时发生的事件，但可以都不发生。例如，事件A和事件B互斥，则有A发生时B不发生，反之亦然。对立事件是特殊的互斥事件，若事件A的对立事件为 $\overline{A}$，表示A不发生。
- **事件和（并）**：事件A和B的和事件表示A或B或两者同时发生，记作 $C = A \cup B$。
- **事件积（交）**：事件A和B的积事件表示A和B同时发生，记作 $C = A \cap B$。
- **事件差**：事件A和B的差事件表示A发生而B不发生，记作 $C = A - B = A \cap \overline{B}$。

这些符号表示事件间的逻辑关系，需用逻辑思维验证其正确性。

## 2、随机变量
随机变量是试验结果的函数，用于将试验结果的描述进行数字化。例如，抛一枚硬币，可以定义随机变量X，正面朝上时 $X = 1$，反面朝上时 $X = 0$。根据可能取值的性质，随机变量分为两类：离散型和连续型。

### （1）离散型随机变量
离散型随机变量的常见分布包括：
- **伯努利分布**：描述一次只有两个可能结果的试验（如抛硬币）。
- **二项分布**：描述n次独立重复试验中成功的次数。
- **几何分布**：描述在第一次成功前的失败次数。
- **泊松分布**：描述单位时间或空间内随机事件的发生次数。

### （2）连续型随机变量
连续型随机变量的常见分布包括：
- **正态分布**：描述大量独立随机变量之和的分布，呈钟形曲线。
- **指数分布**：描述事件间隔时间的分布，常用于寿命分析和排队论。

### （3）随机变量的分布
随机变量的分布描述其取各种值的概率情况。人们总结生活中的现象，得出了以下常见分布：

- **离散型分布**：伯努利分布、二项分布、几何分布、泊松分布。
- **连续型分布**：正态分布、指数分布。

随机变量的分布是其概率性质的完整刻画，而其数字特征（如期望、方差、协方差）则由分布决定，刻画了随机变量的某一方面的性质。
