## 一、PyTorch简介

PyTorch 是由 Facebook AI 研究团队开发的一个深度学习框架。它以其简洁明了的编程风格和易于调试的特点受到许多研究者的喜爱。

PyTorch 的一个重要特点是它的动态计算图，这使得我们可以在运行时更改网络的行为。这对于复杂的网络结构和实验性的项目特别有用。另外，PyTorch 提供的 Autograd 系统能自动计算梯度，这对于实现神经网络的反向传播非常方便。

## 二、安装配置

PyTorch的安装过程非常简单，具体的步骤取决于你的操作系统和你是否拥有支持CUDA的NVIDIA显卡。以下是在具有CUDA支持的环境中安装PyTorch的步骤，包括Windows，macOS，和Linux。

在开始之前，你需要确保你已经安装了Python和pip。PyTorch需要Python 3.x版本，推荐使用Python 3.7或者更高版本。你也可以选择在一个特定的Python虚拟环境中安装PyTorch。

**Step1：**打开终端窗口。

**Step2：**输入以下命令来安装PyTorch：

```bash
pip install torch torchvision torchaudio
```

注意：以上的安装命令适用于支持CUDA的环境，如果你的环境不支持CUDA，或者你想要安装CPU-only版本的PyTorch，你可以访问PyTorch官方网站的[安装页面](https://pytorch.org/get-started/locally/)，选择对应的配置来获取正确的安装命令。

安装完成后，你可以通过以下Python代码来验证PyTorch是否安装成功：

```python
import torch
print(torch.__version__)
```

如果安装成功，这段代码将打印出你安装的PyTorch版本号。

## 三、动态图和静态图

深度学习的计算模型通常可以分为两种：静态计算图和动态计算图。这两种计算图主要体现在深度学习框架如TensorFlow（默认静态图）和PyTorch（默认动态图）的设计上。其区别主要体现在计算模型的构建和执行方式上。

### 1、静态计算图

静态图也称为定义与运行分离模式（define-and-run），即在计算任何实际数值之前，需要先定义出完整的计算图。这个计算图代表了数据以及其之间的计算关系，由节点（代表数据）和边（代表操作）组成。这种方式的主要优势在于可以进行全局优化，如在计算图创建完全之后，系统可以自动地进行内存分配优化、计算任务的并行化、图的剪枝等优化。这使得静态图在大型网络和复杂计算任务时能提供更高的运行效率。

然而，静态图的缺点在于灵活性相对较差。比如，它无法直接处理动态的数据结构，如变长的序列输入等。另外，由于在编程时无法获得中间结果，所以调试相对复杂。

### 2、动态计算图

动态图也称为定义与运行合一模式（define-by-run），即在定义计算过程的同时进行运行。这种方式下，每一步操作都会立即执行并返回结果。动态图的主要优势在于其高度的灵活性，可以方便地实现动态网络结构，比如循环神经网络（RNN）就可以非常自然地表示。此外，由于每一步操作都会立即返回结果，所以更容易进行调试。

但动态图的缺点在于可能会缺乏全局的优化，因为在每一步操作之前，系统无法得知更多的上下文信息，从而进行优化。这在某些情况下，可能会使得计算效率略低于静态图。

值得注意的是，现在许多深度学习框架，如TensorFlow 2.0和PyTorch，已经支持在动态图和静态图之间切换，以便在需要灵活性和计算效率之间进行权衡。

## 四、深度学习库能做什么？

### 1、GPU加速

```python
import torch
import time

print(torch.__version__)
print(torch.cuda.is_available())
# print('hello, world.')


a = torch.randn(10000, 1000)
b = torch.randn(1000, 2000)

t0 = time.time()
c = torch.matmul(a, b)
t1 = time.time()
print(a.device, t1 - t0, c.norm(2))

device = torch.device("cuda")
a = a.to(device)
b = b.to(device)

t0 = time.time()
c = torch.matmul(a, b)
t2 = time.time()
print(a.device, t2 - t0, c.norm(2))

t0 = time.time()
c = torch.matmul(a, b)
t2 = time.time()
print(a.device, t2 - t0, c.norm(2))
```

### 2、自动求导

```python
import torch
from torch import autograd


x = torch.tensor(1.0)
a = torch.tensor(1.0, requires_grad=True)
b = torch.tensor(2.0, requires_grad=True)
c = torch.tensor(3.0, requires_grad=True)

y = a**2 * x + b * x + c

print("before:", a.grad, b.grad, c.grad)
grads = autograd.grad(y, [a, b, c])
print("after :", grads[0], grads[1], grads[2])
```
