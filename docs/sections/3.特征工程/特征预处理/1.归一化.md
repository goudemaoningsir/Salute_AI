## 1、介绍

归一化（Normalization）首先在特征（维度）非常多的时候，可以防止某一维或某几维对数据影响过大，也是为了把不同来源的数据统一到一个参考区间下，这样比较起来才有意义，其次程序可以运行更快。 例如：一个人的身高和体重两个特征，假如体重50kg，身高175cm,由于两个单位不一样，数值大小不一样。如果比较两个人的体型差距时，那么身高的影响结果会比较大。

归一化是数据预处理的重要步骤之一，目的是将数据缩放到特定范围，从而提高模型训练的效果。

## 2、最小-最大归一化 (Min-Max Normalization)

### （1）介绍

最小-最大归一化通过将数据线性变换到一个指定的范围（通常是0到1）来缩放数据。它保留了原始数据的分布和关系，适用于数据分布不明显偏离均匀分布的情况。公式如下：

$$
 x' = \frac{x - x_{\min}}{x_{\max} - x_{\min}} 
$$

这里，$ x $ 是原始数据。$ x' $ 是归一化后的数据，$ x_{\min} $ 是数据的最小值，$ x_{\max} $ 是数据的最大值。通过这个公式，数据被缩放到0到1之间。

### （2）代码示例

```python
import numpy as np

# 生成示例数据
data = np.array([1, 2, 3, 4, 5])

# 最小-最大归一化函数
def min_max_normalize(data):
    min_val = np.min(data)
    max_val = np.max(data)
    return (data - min_val) / (max_val - min_val)

# 应用最小-最大归一化
normalized_data = min_max_normalize(data)
print("Min-Max Normalized Data:", normalized_data)
```

### （3）总结

**适用场景：** 当数据分布有明显的边界时，且数据不受异常值影响时，最小-最大归一化是一个很好的选择。例如，图像处理中像素强度范围通常在0到255之间，可以使用最小-最大归一化将其缩放到0到1之间。

**注意事项：** 最小-最大归一化对异常值敏感，可能会受到极端值的影响。

## 3、Z-Score 归一化 (Standardization)

### （1）介绍

Z-Score 归一化（标准化）通过将数据转换为均值为0、标准差为1的分布来缩放数据。它适用于数据服从高斯分布（正态分布）的情况。公式如下：
$$
x' = \frac{x - \mu}{\sigma}
$$

其中，$ x $ 为原始数据，$ x' $ 是归一化后的数据，$ \mu $ 是数据的均值，$ \sigma $​ 是数据的标准差。通过这个公式，数据被转化为标准正态分布。

### （2）代码示例

```python
import numpy as np

# 生成示例数据
data = np.array([1, 2, 3, 4, 5])

# Z-Score 归一化函数
def z_score_normalize(data):
    mean = np.mean(data)
    std_dev = np.std(data)
    return (data - mean) / std_dev

# 应用 Z-Score 归一化
normalized_data = z_score_normalize(data)
print("Z-Score Normalized Data:", normalized_data)
```

### （3）总结

**适用场景：** 当数据分布接近正态分布时，Z-Score 归一化是一个很好的选择。它可以使数据的均值为0，方差为1，适用于许多机器学习算法（如线性回归、逻辑回归等）。

**注意事项：** Z-Score 归一化假设数据近似正态分布，因此在数据不满足这个假设时效果可能不佳。

## 4、小数定标归一化 (Decimal Scaling Normalization)

### （1）介绍

小数定标归一化通过将数据按10的幂次缩放，使数据的最大绝对值小于1。这种方法简单直接，适用于数据的最大值和最小值已知的情况。公式如下：
$$
x' = \frac{x}{10^j}
$$

其中，$ x $ 为原始数据，$ x' $ 是归一化后的数据，$ j $​ 是使数据的最大绝对值小于1的最小整数。通过这个公式，数据被缩放到[-1, 1]之间。

### （2）代码示例

```python
import numpy as np

# 生成示例数据
data = np.array([1, 2, 3, 4, 5])

# 小数定标归一化函数
def decimal_scaling_normalize(data):
    max_val = np.max(np.abs(data))
    j = np.ceil(np.log10(max_val))
    return data / (10 ** j)

# 应用小数定标归一化
normalized_data = decimal_scaling_normalize(data)
print("Decimal Scaling Normalized Data:", normalized_data)
```

### （3）总结

**适用场景：** 小数定标归一化适用于数据的最大值和最小值已知，并且希望将数据缩放到[-1, 1]或[0, 1]之间的情况。它是一种简单有效的归一化方法。

**注意事项：** 小数定标归一化的缩放取决于数据的最大绝对值，因此对于存在极端值的数据可能不够稳健。

## 5、最大绝对值归一化 (Max Absolute Normalization)

### （1）介绍

最大绝对值归一化通过将数据的每个值除以数据的最大绝对值来缩放数据，使数据的范围在[-1, 1]之间。它保留了数据的稀疏性，适用于稀疏数据的情况。公式如下：
$$
x' = \frac{x}{\max(|x|)}
$$

其中，$ x $ 为原始数据。$ x' $ 为归一化后的数据。$ \max(|x|) $​ 为数据的绝对值的最大值。通过这个公式，数据被缩放到[-1, 1]之间。

### （2）代码示例

```python
import numpy as np

# 生成示例数据
data = np.array([1, 2, 3, 4, 5])

# 最大绝对值归一化函数
def max_absolute_normalize(data):
    max_val = np.max(np.abs(data))
    return data / max_val

# 应用最大绝对值归一化
normalized_data = max_absolute_normalize(data)
print("Max Absolute Normalized Data:", normalized_data)
```

### （3）总结

**适用场景：** 最大绝对值归一化适用于稀疏数据，可以保留数据的稀疏性，并将数据缩放到[-1, 1]之间。它在处理稀疏数据时比较有用。

**注意事项：** 最大绝对值归一化可能会放大噪声和异常值，因此在数据中存在大量噪声或异常值时需要谨慎使用。

## 6、测试数据如何归一化

在训练机器学习模型时，通常会将训练数据归一化，然后将相同的归一化变换应用于测试数据。这样可以确保训练数据和测试数据在同一尺度上，从而提高模型的性能和稳定性。

步骤如下：

**Step1：计算训练数据的统计信息**：根据训练数据计算所需的统计信息（如最小值、最大值、均值、标准差等）。

**Step2：归一化训练数据**：使用计算出的统计信息对训练数据进行归一化。

**Step3：使用训练数据的统计信息归一化测试数据**：使用训练数据的统计信息对测试数据进行相同的归一化变换。

以下是对训练数据和测试数据进行最小-最大归一化和Z-Score归一化的详细代码示例。

```python
# 最小-最大归一化 (Min-Max Normalization)
import numpy as np

# 生成示例训练数据和测试数据
train_data = np.array([1, 2, 3, 4, 5])
test_data = np.array([2, 3, 4, 5, 6])

# 最小-最大归一化函数
def min_max_normalize(train_data, test_data):
    min_val = np.min(train_data)
    max_val = np.max(train_data)
    train_normalized = (train_data - min_val) / (max_val - min_val)
    test_normalized = (test_data - min_val) / (max_val - min_val)
    return train_normalized, test_normalized

# 应用最小-最大归一化
train_normalized, test_normalized = min_max_normalize(train_data, test_data)

print("Min-Max Normalized Train Data:", train_normalized)
print("Min-Max Normalized Test Data:", test_normalized)
```
