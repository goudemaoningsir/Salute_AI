## 一、概述

Scikit-learn（简称sklearn）是一个基于Python的开源机器学习库，它建立在NumPy、SciPy和matplotlib这些科学计算库之上，提供了一系列强大的工具，用于机器学习和统计建模，包括分类、回归、聚类和降维等。

sklearn的主要特点包括：

- **算法多样性**：提供了广泛的机器学习算法，如支持向量机（SVM）、随机森林、k-近邻（KNN）、朴素贝叶斯、逻辑回归等。
- **统一的接口**：sklearn的API设计简洁一致，使得不同算法之间的切换和使用变得容易。
- **可扩展性**：允许用户自定义模型和算法，并且可以轻松地与其他Python库集成。
- **数据预处理**：提供了数据标准化、归一化、特征选择等预处理功能。
- **模型评估**：支持交叉验证、性能度量等模型评估工具。
- **管道**：可以创建数据处理和模型训练的管道，简化工作流程。

## 二、安装
sklearn可以通过Python的包管理器pip进行安装。以下是安装sklearn的基本命令：

```bash
pip install -U scikit-learn
```

如果你使用的是Anaconda环境，可以使用conda进行安装：

```bash
conda install -c conda-forge scikit-learn
```
